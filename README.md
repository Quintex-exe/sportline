# FOOTINT — Global Football Intelligence

FOOTINT is a futuristic football intelligence visualization platform that combines live-style event feeds, geospatial mapping, animated transfer routes, and cyberpunk-inspired UI design into a single interactive experience.

The project visualizes football activity around the globe using a rotating 3D Earth interface powered by `globe.gl` and `three.js`.

---

# Features

## Interactive Globe

* Rotating 3D Earth visualization
* Atmospheric glow effects
* Night Earth texture
* Animated football intelligence arcs
* Smooth auto-rotation and drag controls

## Intelligence Feed

* Live simulated football activity feed
* Transfer alerts
* Signing updates
* Injury reports
* Scouting activity
* Match intelligence events

## Cyberpunk UI

* Futuristic terminal aesthetic
* Scanline effects
* Glassmorphism panels
* Neon glow styling
* Responsive layout

## Real-Time Simulation

* Dynamic arc generation
* Live updating events
* UTC clock synchronization
* Animated node activity
